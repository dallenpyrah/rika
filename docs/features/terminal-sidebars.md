# Terminal sidebars

The left Thread sidebar lists Threads and can be opened, focused, navigated, and used to switch the active Thread. The right sidebar shows either the Workspace file tree or changed files; those two views are mutually exclusive.

Each sidebar scrolls independently. Open sidebars take space from the transcript and composer instead of covering them, and their width is bounded to keep the main terminal usable. Users may drag the changed-files sidebar edge within those bounds.
