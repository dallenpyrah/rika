# Transcript navigation and detail

The transcript follows new output while the user is at the bottom. Scrolling or Page Up pauses follow without moving the reading position as output streams; End returns to the live bottom. Reaching a mounted edge pages through older or newer transcript entries while preserving the visible anchor.

Tab and Shift+Tab move through expandable reasoning, tool, diff, and subagent rows. Enter or a click toggles the selected row. The terminal mounts at most 200 semantic entries at once.
