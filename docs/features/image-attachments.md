# Image attachments

Users can add supported images by terminal paste, clipboard paste, dropped paths, bracketed paths, and file URLs. The composer shows each image as a numbered attachment while preserving its position among text parts.

Images remain structured prompt parts through submission, persistence, and replay. Rika validates pasted bytes and media type, stores accepted terminal images under the Workspace, and removes only an attachment that fails to materialize. Images cannot be added while editing a queued prompt.
