# Input and control

The composer supports multiline editing, history, external editing, structured image attachments, collapsed pasted text, file and Thread mentions, queue editing, steering, interruption, shell prefixes, and explicit submission failures that preserve the draft.

Image bytes remain structured prompt parts through persistence and replay. Collapsed paste tokens are composer-only and expand to exact text before transcript, persistence, or model input.

The command palette contains Thread switching, mode selection, fast-mode toggle, and quit. The Thread browser provides transcript preview. The left Thread sidebar and right changed-files or Workspace sidebar are independently scrollable and resize the transcript rather than covering it.
